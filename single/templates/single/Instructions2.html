Anlagemöglichkeiten

Der Finanzberater teilt das gesamte Vermögen des Kunden beliebig zwischen folgenden beiden Anlagemöglichkeiten auf:

Die sichere Anlage

Die riskante Anlage

 

Der Kunde erhält eine Rendite von 0% auf das investierte Kapital.

 

Mit einer Wahrscheinlichkeit von  erhält der Kunde eine Rendite von +250% auf das investierte Kapital.

Mit einer Wahrscheinlichkeit von  erhält der Kunde eine Rendite von -100% auf das investierte Kapital.

 

Für den Kunden setzt sich die Auszahlung aus der Geldanlage wie folgt zusammen:

Bei einem guten Ausgang der riskanten Anlage:
Auszahlung = {{ Constants.endowment_principals }} - Investment + (3,5 * Investment)

Bei einem schlechten Ausgang der riskanten Anlage:
Auszahlung = {{ Constants.endowment_principals }} - Investment
